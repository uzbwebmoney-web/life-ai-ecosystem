| # | Модуль | Подмодули MVP |
|---|--------|---------------|
| 16 | AI-чат | вопросы, перевод, документы |
| 18 | Календарь | встречи, напоминания, ДР |
| 19 | Уведомления | лекарства, подписки, кредиты |
| 3 | Финансы | доходы, расходы, бюджет |

## Фаза 4 — Расширение

Юридическая помощь, бизнес, работа, образование, путешествия, госуслуги, безопасность, экстренная помощь…

## Фаза 5 — Экосистема

- Семейный режим (общий календарь **внутри бота**, без синхронизации с Google/Apple)
- Голосовой режим end-to-end (Whisper + TTS)
- Умный дом, новости, развлечения

**Не планируется:** OAuth/API Google Calendar, почта, облачные хранилища, банковские push — только ручной ввод и напоминания из SQLite.

## Критерий «готово к запуску MVP»

- Пользователь может: записать расход, поставить напоминание, загрузить чек, спросить AI, найти «когда менял масло» (если внёс данные).
- Семейный профиль — опционально, один пользователь достаточен для beta.
