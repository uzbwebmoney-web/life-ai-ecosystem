# Life AI Ecosystem

Персональная AI-экосистема в Telegram — **14 разделов** + cross-cutting возможности.

## Запуск

```bash
cd "C:\Users\Windows 11\life-ai-ecosystem"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
python -m app.main
```

`.env`:
```
BOT_TOKEN=...
OPENAI_API_KEY=...   # AI, голос (Whisper/TTS), OCR (Vision)
```

## Реализовано

### 14 разделов
Здоровье, авто, финансы, бизнес, право, путешествия, дом, покупки, образование, питание, фитнес, органайзер, AI-ассистент, сейф.

### Cross-cutting
| Функция | Как |
|---------|-----|
| 🏠 Главный экран | Приветствие, «Сегодня», погода, SOS |
| 🔔 Уведомления | Единый центр: кредиты, счета, лекарства, подписки |
| 🤖 AI-чат | `/ask`, свободный текст, proactive-кнопки |
| 🧠 Память AI | «Запомни: …», вкл/выкл в настройках |
| 🔍 Поиск | Записи + память (семейный доступ учитывается) |
| 🎤 Голос | Whisper + TTS при включённом голосовом режиме |
| 📷 Сканер | Универсальный — с предложением записать расход |
| 👨‍👩‍👧 Семья | Профили + семейный доступ `/join КОД` |
| 🚨 SOS | Срочная помощь (RU/UZ/EN) |

### Кредит и ипотека
Финансы → **Кредит и ипотека** или `/credit`:
1. Название
2. Общая сумма
3. Ежемесячный платёж
4. День оплаты (1–31)

Бот ведёт **остаток долга**. В день оплаты приходит напоминание с кнопкой «Записать платёж» — вводите сумму, остаток пересчитывается автоматически.
